about=This plugin provides functionality to convert between NASA's Unified Metadata Model (UMM) and STAC (SpatioTemporal Asset Catalog) with the liability-claims extension. It supports bidirectional conversion, allowing users to import UMM data into STAC format and export STAC data to UMM format. The plugin includes full support for quality information conversion, mapping UMM DataQuality fields to ISO 19115-like quality reports in STAC's liability:quality field.
